Here's how to do the next release:

Update the version number in this file
Update CHANGELOG.txt
Update README.txt to reflect current capabilities
Make sure the tests run
Check everything in

cd ~/data/
cvs -q rtag -D tomorrow "ruby_doom_release_0_3" ruby-doom
mkdir ruby-doom-0.3
cp ruby-doom/etc/README.txt ruby-doom-0.3/
cp ruby-doom/etc/CHANGELOG.txt ruby-doom-0.3/
cp ruby-doom/etc/LICENSE.txt ruby-doom-0.3/
cp ruby-doom/src/ruby/*.rb ruby-doom-0.3/
cp ruby-doom/test_wads/simple.wad ruby-doom-0.3/
zip -q -r ruby-doom-0.3.zip ruby-doom-0.3/
rm -rf ruby-doom-0.3/

Go to Admin, Edit/Release Files, click on Add new release
Paste stuff into the changelog/readme boxes
Add the ruby-doom-0.3.zip file

Submit news on RubyForge and Freshmeat
